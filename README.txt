# Blossom Craft Notifier

A free, community-focused timer tool designed specifically for the Blossom Craft community.

---

## Features

- Multiple preset timer modes tailored for Blossom Craft activities  
- Custom timer durations to fit your unique needs  
- Right-click mouse resets to quickly restart timers  
- Sound alerts and desktop notifications with easy toggles  
- Mouse listener to enable/disable right-click detection  
- Progress bar showing time left on the timer  
- History tab logs the last 50 timer events with timestamps  
- Counts and displays right-clicks in the last 60 minutes  
- Light and dark themes for your preferred look  
- Always-on-top window keeps the notifier visible  
- Simple, user-friendly interface  

---

## Installation

1. Download the latest Windows executable (`.exe`) from the official source.  
2. No additional setup or dependencies needed.  
3. Run the `.exe` file to launch the notifier.

---

## Usage

1. Select a timer mode from the dropdown or enter a custom time in seconds.  
2. Click **Start** to begin the timer.  
3. Use right-click to reset the timer if mouse listener is enabled.  
4. Toggle sound, notifications, and mouse listener as needed.  
5. View recent timer events and right-click count in the History tab.  
6. Switch between light and dark themes for comfort.
